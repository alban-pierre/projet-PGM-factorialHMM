Projet du cours PGM sur les factorial HMM.

Les commandes github utiles :
git clone url nomdudossier (pour créer le projet)
git add fichierAUpdater.m
git commit -m "message du commit en une seule ligne"
git rm fichierASupprimerDuProjet.m
git pull (pour se mettre à jour par rapport au "serveur")
git push (pour mettre à jour ses commits sur le "serveur")

Autres commandes utiles :
git status (pour voir les différences entre le projet actuel, le commit et le "serveur")
git reset fichier.m (inverse de git add : si on a fait git add à ce fichier et que finalement on ne veux pas inclure les modifications de ce fichier dans le commit)

Toujours faire un pull avant de faire un push (et ainsi régler les problèmes de fusion de fichiers sur son pc avant de faire le push)
